# Aurora World Engine
# 01_ゲームデザイン設計書

---

# 1. ゲームコンセプト

Aurora World Engine は、
プレイヤー自身が世界を作り、
AIゲームマスターと共に物語を体験する
AIストーリーゲームエンジンである。

プレイヤーは

・世界を作る

・キャラクターを作る

・自分（ペルソナ）を作る

・物語を始める

・途中で世界を変化させる

ことができる。

ゲームはSessionとして保存され、
いつでも続きを遊ぶことができる。

---

# 2. メインメニュー

Aurora World Engineは4つの画面で構成される。

・PLAY

・CREATE

・CHARACTER LIBRARY

・SETTINGS

---

# 3. PLAY

PLAYはゲームを遊ぶ画面である。

PLAYは

・新しく始める

・続きから

の2つで構成される。

---

## 新しく始める

Project一覧を表示する。

Projectを選択すると
開始準備画面へ移動する。

---

### 開始準備画面

開始前に

・Project概要

・Scenario

・登場Character

・Persona

を設定する。

その後

「プレイ開始」

を押すと

新しいSessionを作成する。

そのSessionでゲームが始まる。

---

## 続きから

保存されているSession一覧を表示する。

Sessionを選択すると
前回の続きを開始する。

---

# 4. Session

Sessionはゲームのセーブデータである。

Sessionには

・Project

・Scenario

・Persona

・登場Character

・チャット履歴

を保存する。

Sessionはプレイ中に自動保存される。

---

# 5. Chat

Chat画面では

AIゲームマスターと会話しながら
物語を進める。

チャット中でも

・Character追加

・Persona変更

ができる。

---

# 6. CREATE

CREATEは世界を作る画面である。

---

## 新規作成

・Project

・Character

・Persona

---

## 編集

・Project

・Character

・Persona

編集では

・編集

・削除

ができる。

削除は必ず確認画面を表示する。

---

# 7. CHARACTER LIBRARY

Character Libraryには
すべてのCharacterを保存する。

Characterは

・Project Character

・単独Character

を区別せず管理する。

Character Libraryでは

・編集

・削除

・Character Chat

が行える。

---

# 8. Character Chat

Character Libraryから

Characterを選択すると

そのCharacter単体とのチャットを開始できる。

Character Chatは

Projectとは独立したモードである。

---

# 9. Persona

Personaはプレイヤー自身である。

Personaは

・開始前

・プレイ中

どちらでも変更できる。

---

# 10. ゲームの基本思想

Aurora World Engineは

AIチャットではない。

プレイヤーが作った世界を

長期間育てていく

ストーリーゲームエンジンである。

世界はProjectとして保存され、

物語はSessionとして積み重なっていく。
