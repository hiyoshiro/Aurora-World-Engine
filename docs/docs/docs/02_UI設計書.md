# Aurora World Engine
# 02_UI設計書

---

# 1. UIコンセプト

Aurora World Engine は

「作る画面」と
「遊ぶ画面」

を完全に分離する。

作成画面は

シンプル
見やすい
編集しやすい

ことを重視する。

プレイ画面は

没入感

スマホ操作

世界観

を重視する。

---

# 2. メインメニュー

ホーム画面には

・PLAY

・CREATE

・CHARACTER LIBRARY

・SETTINGS

のみ表示する。

迷わないUIを目指す。

---

# 3. PLAY

PLAYは

・新しく始める

・続きから

の2画面で構成する。

---

## 新しく始める

Project一覧を表示する。

カード形式で表示。

カードには

・画像

・Project名

・概要

を表示する。

Projectを押すと

開始準備画面へ移動する。

---

## 開始準備画面

表示項目

・Project画像

・Project説明

・Scenario選択

・登場Character選択

・Persona選択

・プレイ開始ボタン

プレイ開始時に
Sessionを新規作成する。

---

## 続きから

Session一覧を表示する。

一覧には

・Session名

・Project名

・最終更新日時

・最後の会話

・サムネイル

を表示する。

Sessionを押すと
チャット画面へ移動する。

---

## Chat画面

背景にProject画像を表示する。

半透明のチャット表示。

スマホ操作を最優先。

チャット中でも

・Character追加

・Persona変更

・Session設定

を開ける。

---

# 4. CREATE

CREATEは

新規作成

編集

で構成する。

---

## 新規作成

・Project

・Character

・Persona

---

## 編集

・Project

・Character

・Persona

一覧から選択し

編集

削除

ができる。

---

# 5. Character Library

Character一覧を表示。

カード表示。

表示項目

・画像

・名前

・説明

選択後

・編集

・削除

・Character Chat

ができる。

---

# 6. SETTINGS

設定画面は

AI

表示

画像

データ管理

に分ける。

---

# 7. 画像設定

Project画像

Character画像

を設定できる。

画像選択時は

推奨サイズを表示する。

保存前に

・切り抜き

・拡大縮小

・位置調整

を行う。

保存後のプレビューを表示する。

画像が無い場合

AI画像生成

を選択できる。

---

# 8. スマホUI

Auroraは

スマホで遊ぶことを前提に設計する。

重要な操作は

親指だけで届く位置に配置する。

チャット入力欄は常に画面下部。

メニューは片手操作を考慮する。

---

# 9. デザイン方針

黒を基調としたダークテーマ。

Project画像を世界観として利用する。

キャラクター画像は没入感を高める目的で使用する。

画面遷移はできるだけ少なくし、

ユーザーが迷わないUIを目指す。
